## 📌 Sobre o Projeto

Este repositório contém uma aplicação clássica de *To-Do List* desenvolvida com JavaScript Vanilla. O objetivo principal do projeto é demonstrar a manipulação direta do DOM (Document Object Model), tratamento de eventos e a criação de interfaces modernas e responsivas utilizando frameworks de utilitários CSS.

A interface foi inteiramente construída no padrão *Dark Mode*, garantindo conforto visual e um design minimalista.

## 🚀 Funcionalidades

* **Adição de Tarefas:** Inserção dinâmica de novos itens na lista através do input de texto.
* **Validação de Dados:** Prevenção contra a criação de tarefas vazias, exigindo o preenchimento dos dados.
* **Exclusão Rápida:** Botões de exclusão que aparecem interativamente ao passar o mouse (*hover state*) sobre uma tarefa.
* **Notificações Visuais (Toasts):** Feedback em tempo real para o usuário (sucesso ao criar/excluir e erro ao tentar enviar vazio) utilizando a biblioteca Toastify.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica.
* **JavaScript (Vanilla):** Lógica de programação, manipulação do DOM e criação dinâmica de elementos HTML.
* **Tailwind CSS (via CDN):** Estilização rápida, prototipagem e classes utilitárias diretamente na marcação.
* **Toastify JS:** Biblioteca de notificações elegantes e não-intrusivas.
